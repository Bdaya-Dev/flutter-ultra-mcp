# @flutter-ultra/flutter-ultra-gesture

MCP server for **gesture dispatch** inside a running Flutter app via the in-app `ultra_flutter` mixin: tap, enter text, scroll, screenshot, interactive-element enumeration.

**Status:** scaffold stub. Implementation owner: **wave-2 gesture worker** (see plan §12).

Coordinates with the runtime server via `state/sessions.json` (read-only consumer).
